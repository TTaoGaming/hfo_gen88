### The Weapon: Stryker Mutation Testing

Unit tests are not enough. **Mutation Testing** is the only way to prove your tests are not "Theater."

- **Action**: Red Regnant injects small bugs (mutants) into the code (e.g., changes `true` to `false`, `>` to `>=`).

- **Failure**: If the tests **PASS**, the code is "Rotten" (it didn't notice the change). This is what we call **Automation Theater** or **Reward Hacking**.
